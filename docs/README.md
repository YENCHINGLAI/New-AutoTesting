# New AutoTesting

## 中文 | [English](README_EN.md)

![Release](https://img.shields.io/badge/Python-3.11.6%2B-blue)

- 新自動化測試主程式 (支持windows、Linux、macOS), UI使用Qt

## 功能
- 主要目的為, 涵蓋所有型態之產品自動化測試
- 執行YAML測試腳本, 並記錄db及生成html report

## 使用方式
- 1.下載最新版本 
- 2.解壓縮.7z
- 3.執行AutoTesting.exe
